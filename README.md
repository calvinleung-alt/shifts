# Shifts Management

### Client 

#### Start Application
```
cd ./client
npm install
npm run start
```

Open browser and check `localhost:3000`

### Server

#### Start Application
```
cd ./server
npm install
npm run start
```

#### Test Application
```
cd ./server
npm run test
```

### Notes
State of the data will be persisted after confirming/declining the shift, 
restart the server application can help resetting the state.
